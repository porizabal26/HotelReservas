# Sistema de Reservas de Hotel

Este proyecto es una aplicación de gestión de reservas para un hotel.
Permite registrar nuevas reservas, listar habitaciones disponibles y consultar las reservas activas de un cliente.

## Requisitos

Antes de ejecutar este proyecto, asegúrese de tener instalados los siguientes componentes:

- [Visual Studio](https://visualstudio.microsoft.com/)
- [.NET SDK](https://dotnet.microsoft.com/download)
- [SQL Server](https://www.microsoft.com/sql-server)
- [SQL Server Management Studio (SSMS)](https://docs.microsoft.com/es-es/sql/ssms/download-sql-server-management-studio-ssms)

Los scripts SQL con la creación de tablas, consultas y el procedimiento almacenado se encuentran en el archivo **/database-scripts.sql**
